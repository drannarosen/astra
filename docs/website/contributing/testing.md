# Testing

ASTRA's early test suite is allowed to be scaffold-oriented, but it still has to protect real invariants.

## Early testing priorities

- import and package integrity,
- state and grid consistency,
- callable microphysics interfaces,
- residual and Jacobian dimensions,
- docs presence and site structure,
- runnable examples.

As ASTRA matures, tests should shift steadily toward physical claims and benchmark comparisons.
