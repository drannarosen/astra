# Entropy-DAE

Entropy-DAE is a major scientific direction for ASTRA, but it is intentionally a **later** milestone.

The current bootstrap includes the type and documentation slot for `EntropyDAEFormulation` so that the framework acknowledges the method without letting it dominate the repository identity too early.

This separation is not administrative. It is scientific discipline.

Entropy-DAE should also remain a later differentiability milestone. ASTRA should first learn how to differentiate a converged classical solve cleanly, then promote that derivative discipline into a richer differential-algebraic time-evolution frontend.
