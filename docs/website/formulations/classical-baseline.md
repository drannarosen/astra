# Classical Baseline

The classical baseline is ASTRA's canonical lane.

This is the formulation that must become trustworthy first because it provides:

- the first serious forward-solver benchmark,
- the reference lane for validation,
- the first explicit derivative boundary for a converged structure solve,
- and the baseline against which later alternative formulations are judged.

If ASTRA cannot support a clean classical lane, it is not yet ready to be a formulation laboratory.
