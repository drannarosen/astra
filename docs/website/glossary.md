# Glossary

## ASTRA

Adaptive STellar Research Architecture.

## Classical baseline

ASTRA's canonical first serious formulation: a clean forward stellar-structure lane that becomes the reference surface for later method comparisons.

## Entropy-DAE

A later research formulation that ASTRA intends to host once the classical baseline is trustworthy.

## Structure problem

The immutable problem bundle combining formulation, parameters, composition, grid, microphysics, and solver settings.

## Structure diagnostics

The solve summary object that reports convergence information and a small set of interpretable scalar outputs.
